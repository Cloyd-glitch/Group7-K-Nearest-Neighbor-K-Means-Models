# Group7-K-Nearest-Neighbor-Predictive-Modeling
Predictive modeling using the K-Nearest Neighbors algorithm, applied to medical information like the Pima Indians Diabetes Dataset, represents a foundational workflow in applied machine learning.
K-Means clustering is an unsupervised machine learning algorithm designed to partition a dataset into 
 distinct, non-overlapping groups (clusters). It groups data points by minimizing the distance between each point and its assigned cluster's center, known as a centroid.
